# fresh-ubuntu

script for installing all my required tools when a fresh installation is done.

## How to install 
For installing the essential tools run 
```bash
bash run.sh  
```
For installing the general tools run
```bash
bash general.sh  
```

## Tools essential

- Joplin
- Bitwarden
- Sublime Text
- VS Code
- Typora
- Brave browser
